How to Add New Recipes

To add a new recipe to this collection:

  1. Create an HTML file with proper Schema.org Recipe JSON-LD markup
  2. Save the file in the recipes/ directory
  3. Commit and push to the repository
  4. The recipe will be automatically available at https://traviscaro.github.io/recipe-importer/recipes/[filename].html

Note: Ensure your HTML files include proper Recipe JSON-LD structured data for AnyList compatibility.
